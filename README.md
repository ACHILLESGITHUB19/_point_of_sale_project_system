## Introduction
POINT_OF_SALES_SYSTEM is a simple point-of-sale application developed for academic purposes. It allows managing products, categories, brands, and transactions efficiently for small business or school project simulations.



## Features
- Manage products, categories, and brands
- Add, edit, and delete items
- Track sales and transactions
- Simple and user-friendly interface
